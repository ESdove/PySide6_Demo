# QProgressDialog

水平或垂直进度条控件

向用户展示程序仍在运行没有卡死，改善用户体验

## Qt 官方文档

[Qt 5.15官方文档-QProgressDialog](https://doc.qt.io/qt-5.15/qprogressdialog.html)



## 槽函数

| 槽函数   | 说明                   | 备注                                |
| -------- | ---------------------- | ----------------------------------- |
| reset()  | 重置进度条对话框       | 如果开启了autoClose()则对话框被隐藏 |
| cancel() | 重置进度条对话框，如果 |                                     |
|          |                        |                                     |



## 信号

| 信号       | 说明                                               |
| ---------- | -------------------------------------------------- |
| canceled() | 当取消按钮被点击后发送此信号。默认连接到cancel()槽 |
